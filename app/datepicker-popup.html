By default, Ngb Datepicker comes with inbuilt validator and there is no way to turn it off. We had a different requirements around date validation so inbuilt one was not needed.
Therefore, I simply added a directive to override the "NgbInputDatepicker.validate" function to always return "null". Try removing directive from input and type invalid date to see the difference.
